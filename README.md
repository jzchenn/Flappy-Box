## Flappy Box
A flappy bird clone made in Python using Tkinter for **[NCEA Level 3 Digital Technologies Internal (91906: Use complex programming techniques to develop a computer program)](https://www.nzqa.govt.nz/ncea/assessment/view-detailed.do?standardNumber=91906)**
<img width="635" height="508" alt="image" src="https://github.com/user-attachments/assets/621f018d-a696-4c9e-9eea-e227330c690d" />

### How to Play
* **Start:** Press `SPACE` to start the game.
* **Flap:** Keep tapping `SPACE`, `Up Arrow`, `W`, or left-click your mouse (`Button-1`) to flap and stay airborne.
* **Restart:** Press `R` if you crash to restart the game.
* **Objective:** Evade the orange obstacles and don't hit the ground!

### Prerequisites
* **Python 3.x**

### How to Run
1. Clone this repository or download the source code.
* **Option 1 (Double-click):** If Python is configured as your default program for `.py` files, simply double-click the file to launch the game window directly.
* **Option 2 (Terminal):** Open your terminal or command prompt in the project directory and run:
  ```bash
  python "91906 (1).py"
  ```
